Data Visualization: We created a ggplot figure in order for us to see exactly how the relationship between amount of deaths and recoveries intertwinds with one another. My group and I decided that a line plot would be of best fit because line plots are useful to track changes over short and long periods of time.

day_wise %>% 
  ggplot(aes(x = log(Deaths), 
             y = log(Recovered))) + 
  geom_point(alpha = .1, color = 'purple') +
  geom_smooth(method = lm, se = F) +
  theme_bw() +
  theme(legend.position = 'top')

Data Transformation: We used the group by function here in order to group rows that have the same values into summary rows, which then allows us to use the function summary, in order to summarize the statistics provided.

day_wise <- day_wise %>% 
  group_by(Deaths, Recovered) %>% 
  summarise(Recovered_tot = sum(Recovered, na.rm = T),
            Deaths_mean = round(mean(Deaths, na.rm = T), 2),
            .groups = "drop")
